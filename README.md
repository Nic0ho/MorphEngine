# MorphEngine
MorphEngine is a custom, cross-platform 3D game engine built from scratch.

## Used instruments
* **Language:** C, C++
* **Graphics API:** Vulkan
* **Windowing & Input:** GLFW
* **Asset Pipeline:** Assimp
* **Math:** GLM
* **Working environments:** CLion, VS Code, Neovim/Vim

This project is a personal development intended for future academic research and thesis work.
